# linode_panel
本项目是一个采用 flask + alpinejs + tailwind 框架开发的基于 api 的 linode 开机面板  
本项目是一个用于学习 alpinejs + flask 的项目，所以写的很烂（里面一堆bug）

## 如何运行
### 1.从源代码开始
```
git clone https://github.com/52ludama/linode_panel.git
```
```
cd /linode_panel
```
```
pip3 install -r requirements.txt
```
```
python3 main.py
```
